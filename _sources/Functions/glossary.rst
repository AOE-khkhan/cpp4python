Glossary
--------

.. glossary::


    argument
        The data passed to parameter.

    ``const``
        Keyword that makes a variable or value constant and unchanging.

    friend
        A class that can access the private data members of another class it is designated as a friend of.

    function
        A section of code that performs a procedure and is usually named.

    overloading
        Changing what an existing operator or function does.

    parameter
        A variable in a method definition that passes data.

    reference
        A value that indicates a place in a computerâ€™s memory.

    ``void``
        Keyword indicating a function has no return value.
